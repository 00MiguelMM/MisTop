# Gestión del proyecto

Para organizar el desarrollo de la aplicación MisTop he utilizado Trello como herramienta de gestión de tareas.

## Metodología

He seguido un enfoque tipo Kanban, organizando el trabajo en distintas columnas:

- Backlog: tareas que tengo pendientes para más adelante
- Todo: tareas listas para empezar
- In process: tareas en las que estoy trabajando
- Review: tareas terminadas pendientes de revisión
- Done: tareas completadas

## Organización del trabajo

He dividido el proyecto en tareas pequeñas y subtareas para facilitar el desarrollo progresivo.

Primero me he centrado en la preparación del proyecto (setup inicial), y después iré avanzando con el desarrollo del backend y el frontend.
